jQuery Geocomplete

A simple plugin for Google Maps Autocomplete.
View Github

Click the blue buttons to view the example

Fields with Data Attributes 
              $("#txtAddress1").geocomplete({
                fields: "#myForm"
              });

              <div id="myForm">
                <input id="txtAddress1" data-geocomplete="street address" />
                <select data-geocomplete="country"></select>
                ...
              </div>
Fields with Data Attributes
Short Values and Field Object 
              $("#txtAddress2").geocomplete({
                fields: {
                  "#txtAddress2" : "short street address",
                  "#ddlCountry2" : "short country",
                  "#txtState2"   : "short state",
                  ".txtCounty2"  : "short county",
                  ".txtCity2"    : "short city",
                  ".txtZipCode2" : "short zipcode"
                }
              });

              $("#txtAddress5").geocomplete({

Callbacks 
              $("#txtAddress6").geocomplete({
                onChange: function(name, result) {
                  console.log(name, result)
                },
                onNoResult: function(name) {
                  console.log("Could not find a result for " + name)
                }
              });
